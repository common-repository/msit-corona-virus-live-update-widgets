== Description ==

This MSIT Corona Virus Live Update Widgets is basically showing world and Country wise Corona Virus update Result. After install this plugin you will get a new "MSIT Corona Virus Live Update Widgets" widgets on your wordpress admin panel under appearance menu. Then just use it and enjoy free of cost your wordpress blog or website. You can add this widget on your wordpress sidebar or any widget area.

== Installation ==

1. At first Upload this plugin(unziped) to the `/wp-content/plugins/` directory "msit-coronavirus-live-update-widgets.zip" from your wordpress admin panel as a plugin
2. Activate the plugin through the "Plugins" menu in WordPress
3. Now go to the Wordpress Widget area and then use "MSIT Corona Virus Live Update Widget" widget as your widget area.

= Add this Widget =

1. Install "MSIT Corona Virus Live Update Widget" through the WordPress admin menu of Appearance and then use the widget.
